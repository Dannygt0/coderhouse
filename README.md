# RedZone - Fitness App 🚀

Proyecto realizado para la segunda pre-entrega del curso de Desarrollo Web en **Coderhouse**.

## Tecnologías y Herramientas Aplicadas:

* **Bootstrap 5:** Utilizado para la estructura global, Navbar, Modal de descarga y el sistema de columnas base.
* **CSS Grid:** Implementado de forma personalizada en la sección de **Planes de Precios**, utilizando `grid-template-columns: repeat(auto-fit, minmax(300px, 1fr))` para un diseño responsive sin media queries.
* **Flexbox:** Aplicado en la alineación del Hero, el Footer y los componentes internos de las tarjetas (cards) para garantizar una distribución equitativa de los elementos.
* **Box Model:** Uso avanzado de `padding`, `margin`, `border` y `outline` para dar jerarquía visual y espaciado profesional a los formularios e imágenes.
* **Responsive Design:** La página es totalmente adaptable a dispositivos móviles.

## Estructura del Proyecto:

* `index.html`: Página principal con Hero, Funciones y Precios.
* `/pages/contacto.html`: Formulario de contacto con validación básica.
* `/css/style.css`: Estilos personalizados y sobrescritura de variables de Bootstrap.
* `/assets/`: Imágenes y recursos utilizados.
